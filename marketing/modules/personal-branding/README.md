# Personal Branding with AI Module

🚧 **In Development** — v0.1 shipping Q3 2026.

AI-powered skills for building a personal brand and monetizing it — for creators, freelancers, founders, and consultants.

## Planned Skills (v1.0)

- [ ] **Niche Discovery** — intersection of skill, demand, and authentic interest
- [ ] **Content Pillars** — 3–5 pillar framework + content angles + topic bank
- [ ] **Long-form Content** — LinkedIn/Twitter/blog with hook-promise-payoff structure
- [ ] **Short-form Video** — talking-head and B-roll scripts for TikTok/Reels/Shorts
- [ ] **Audience Building** — engagement strategy, DM funnel, community building
- [ ] **Offer Design** — productized service / digital product / cohort course positioning
- [ ] **Email/Newsletter Strategy** — list building, lead magnet, sequence design
- [ ] **Monetization Paths** — coaching, course, consulting, affiliate, sponsorship
- [ ] **AI-Augmented Content Workflow** — production stack with AI as multiplier

## Editions

Both VN and Global editions planned. Global Edition will emphasize creator economy (US/EU); VN Edition will emphasize KOL/KOC ecosystem.
