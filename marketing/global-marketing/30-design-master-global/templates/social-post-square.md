# Template: Social Post Square

## Mode
marketing-day-to-day

## Aspect ratio
1:1

## Use case
IG feed post, FB feed post, LinkedIn feed post. Daily content batch — template-driven, instantly readable, brand-consistent.

## Prompt structure

```
[Subject + composition — simple instantly-readable subject, centered OR rule of thirds, single focal]
[Style + medium — graphic illustration OR minimalist photography, batch-friendly template]
[Palette + textures — brand palette strict for series consistency]
[Background — simple, clean, allows headline overlay if needed]
[Negatives — no text in frame, no busy details, no clichéd icons]
```

## Filled example (AI Income Mastery course tip series, post #3)

```
Social post visual for "AI tip #3 — the CRISPE prompt framework", abstract puzzle pieces arrangement showing 6 interlocking shapes coming together, centered composition with rule of thirds, single focal point of the puzzle assembly metaphor. 1:1 square format optimized for IG feed.

Minimalist flat illustration style, geometric shapes with subtle dimension, clean modern aesthetic, batch-friendly template that can be re-skinned for tip #4, #5, etc.

Palette warm cream #F5EBE0 background, brand purple #6B4E9F primary pieces, gold accent #D4A574 highlight piece, soft shadow for depth. Brand-cohesive for the AI Income Mastery series.

Background soft cream gradient, ample negative space around the puzzle subject to add a headline overlay via Canva/HTML if needed.

Negatives — no text in frame, no busy decorative elements, no clichéd brain/lightbulb icons, no neon colors, no stock illustration aesthetic, no warped geometric shapes.
```
