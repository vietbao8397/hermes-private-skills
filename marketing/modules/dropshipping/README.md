# Dropshipping Module

🚧 **In Development** — v0.1 shipping Q2 2026, v1.0 Q3 2026.

AI-powered skills for building and scaling a dropshipping business end-to-end. Targeted for both Vietnamese (Shopee/Lazada/TikTok Shop VN) and global (Shopify/Amazon) markets.

## Planned Skills (v1.0)

- [ ] **Product Research** — winning product detection, demand validation, profit margin analysis
- [ ] **Supplier Vetting** — Alibaba/1688 scoring, sample protocol, MOQ negotiation
- [ ] **Store Setup** — Shopify/WooCommerce/TikTok Shop conversion-optimized setup
- [ ] **Ad Creative Generation** — UGC video scripts, image ads, conversion copy
- [ ] **Campaign Scaling** — break-even analysis, scaling rules, budget reallocation
- [ ] **Fulfillment Ops** — order routing, customer service templates, returns
- [ ] **Cashflow Forecasting** — daily/weekly cashflow, ad spend pacing, supplier payment
- [ ] **Refund/Dispute Playbook** — PayPal/Stripe dispute handling, return prevention

## Editions

This module will ship with both VN and Global editions, following the same dual-edition pattern as the Marketing module.
